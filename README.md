# Exploring Clustering Algorithms with PyCaret

## Overview:
This project utilizes PyCaret, a low-code machine learning library in Python, to explore clustering algorithms. In this project, we leverage PyCaret's inbuilt dataset population feature to load a dataset and apply three different clustering algorithms: K-Means, K-Means Shift, and Hierarchical Clustering. We analyze the performance of these algorithms for different sizes of clusters and evaluate the effectiveness of various preprocessing techniques.

### Installation:
To run this project, make sure you have Python installed on your system. You can install PyCaret and other required dependencies using the following command:
```bash
pip install pycaret
```

## Usage:
1. Clone this repository to your local machine.
2. Navigate to the project directory.
3. Run the Python script Pycaret.ipnyb.
